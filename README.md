# 요구 사항
- 멀티플렉싱 기반의 Socket , TCP/IP 웹 서버 제작.
- Non-Block 기능의 서버.
- GET / POST / DELETE 기능.
- 금지한 HTTP 메서드 허용X.
- Configure File에 대한 파싱 및 룰 제작.
- Php 해석 기능.
- Upload 기능.
- 서버의 임의 종료 불가.
- Read(recv), Write(send) 기능은 이벤트 발생 이후에만 호출.
